# How to join a contest

<hr>

When you first sign in, you will be brought to your dashboard. To access the contest, select contests in the top menu bar. 

![Navigation bar with Contests tab highlighted]({{site.baseurl}}{{page.images}}/image1.png)

From the contests page, you can click join to enter a contest. Senior teams should join Mock CCC ‘26 Senior and junior teams should join Mock CCC ‘26 Junior. 

![Page that shows currently running contests as well as previous contests]({{site.baseurl}}{{page.images}}/image2.png)

To view problems, select the problems tab on the top menu bar. This will only work after you’ve joined the contest.

![Navigation bar with Problems tab highlighted]({{site.baseurl}}{{page.images}}/image3.png)

Click on any problem title listed in the table to view the problem's details. Note that if you have joined a contest, only problems from that contest will appear on the problems page.

![Problems page for the joined contest]({{site.baseurl}}{{page.images}}/image4.png)
